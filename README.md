### 一、主要组件
以Hadoop为核心，Hadoop大数据应用生态中最主要的组件，在整个大数据应用于研发已经形成了一个基本完善的生态系统。
### 1、HDFS
- HDFS：Hadoop分布式文件系统，通过流式数据访问，提供高吞吐量应用程序数据访问功能，适合大型数据集的应用程序。
